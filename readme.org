* Introduction
This is my personal emacs config that I use for pretty much everything that it's able to do.
I've separated everything into modules that are then loaded in =init.el=, the modules are sorted into folders as follows;
| Folder     | Purpose                                               |
|------------+-------------------------------------------------------|
| completion | Modules related to text/code completion               |
| evil       | Modules related to evil-mode (for vim motions)        |
| keybinds   | Modules related to mapping keybinds                   |
| lsp        | Modules related to lsp, or anything language specific |
| org        | Modules related to org-mode                           |
| projects   | Modules related to managing projects                  |
| theme      | Modules related to configuring the theme and visuals  |
| misc       | Modules that cannot be sorted into any of the above   |

* Setup
** Steps
+ Download the font: [[https://github.com/protesilaos/iosevka-comfy][iosvka comfy]]
+ Clone repo to ~/.emacs.d
#+begin_src shell
git clone https://github.com/m51k/.emacs.d.git
#+end_src
+ Launch emacs
+ Do =M-x all-the-icons-install-fonts= (dired uses these)
